# Onepage de una Asignatura de Upsin | Desarrollador Front-End y diseñador

> Proyecto desarrollado como aplicación unipágina para la asignatura de Aplicaciones Web. Mostrando mis habilidades de Front-end y UX/UI.

---

## Desafío

El desafío principal fue construir una aplicación web one-page que sirviera para mostrar la información acerca de una asignatura dentro de la carrera, asegurandome que cada componente cumpliera con altos estándares de calidad de código y rendimiento.

* **Objetivo de Calidad:** Implementar la estructura usando CSS y una clara modularización del código para garantizar la mantenibilidad y la fácil adaptación a futuros cambios de diseño.
* **Objetivo de UX:** Ofrecer una navegación intuitiva y una carga de contenido optimizada, propia de una aplicación ligera.

---

## Solución y Características Principales

* **Estructura Modular:** El sitio está dividido en secciones lógicas con un código HTML y CSS bien segmentado.
* **Diseño Responsivo (Mobile First):** La maquetación se construyó priorizando la visualización en dispositivos móviles y tabletas, asegurando la compatibilidad total y una experiencia de usuario óptima en cualquier tamaño de pantalla.
* **Semántica HTML:** Uso correcto de etiquetas HTML5 (`<header>`, `<section>`, `<footer>`, etc.) para mejorar la accesibilidad y el posicionamiento en buscadores.
